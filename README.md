# Laundry Service Website

## Project

Laundry Service Website

## Description

This is a simple Laundry Service Website made using HTML, CSS and JavaScript. Users can view laundry services, add services to the cart, check the total amount and book the service by filling the form.

## Technologies Used

- HTML
- CSS
- JavaScript
- EmailJS
- Font Awesome

## Features

- Responsive Website
- Laundry Services List
- Add Item
- Remove Item
- Total Amount Calculation
- Booking Form
- Email Validation
- Phone Number Validation
- EmailJS Integration
- Local Storage
- Newsletter Section
- Footer

## Project Files

Laundry-Service-Website/
│
├── index.html
├── style.css
├── script.js
├── README.md


## How to Run

1. Download the project.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Add services.
5. Fill the booking form.
6. Click **Book Now**.
