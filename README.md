# Laundry Service Website

## Project Overview

This project is a responsive Laundry Service Website developed using HTML, CSS, and JavaScript. The website allows customers to view laundry services, add services to a booking cart, calculate the total amount, and book laundry services through a simple booking form.

The website is designed with a clean user interface and works properly on desktop, tablet, and mobile devices.

## Technologies Used

- CSS3
- JavaScript
- EmailJS

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open **index.html** in any web browser.
4. Browse the website.
5. Add laundry services.
6. Fill the booking form.
7. Click **Book Now**.

## EmailJS Setup

1. Create an EmailJS account.
2. Create an Email Service.
3. Create an Email Template.
4. Copy your:

- Public Key
- Service ID
- Template ID

5. Open **script.js**

Replace:

```javascript
emailjs.init("YOUR_PUBLIC_KEY");
```

Replace:

```javascript
"YOUR_SERVICE_ID"
```

Replace:

```javascript
"YOUR_TEMPLATE_ID"
```

Save the file.

Now booking details will be sent to your email.
